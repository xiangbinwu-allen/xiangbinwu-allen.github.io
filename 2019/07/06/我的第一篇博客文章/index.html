title: 我的第一篇博客文章
date: 2019-07-06 12:37:26
tags:

##第一步: 下载nodejs : nodejs.org
下载下述版本,并按步骤进行安装

##第二步: 使用cmd命令行, 安装cnpm:
cnpm介绍: cnpm为npm淘宝提供npmjs.org 镜像, 用于替换npm命令, 便于资源包下载直接从国内的淘宝服务器下载
npm install -g cnpm -registry=https://registry.npm.taobao.org

查看:cnpm版本号: cnpm -v

第三步: 使用cnpm安装hexo框架
命令: cnpm install -g hexo-cli

查看:hexo版本号: hexo -v

第四步: 初始化hexo
建立一个blog空目录, 建议使用命令行: mkdir “D:Program Files”/blog
进入该目录, 初始化:
hexo init (非管理员用户: sudo hexo init)
如果有问题的,删除文件夹中的内容,重新执行命令.

第五步: 启动hexo
启动服务: hexo s

通过http://localhost:4000 访问本地hexo博客

第六步: 新建自己的一个博文
命令: hexo n “我的第一篇博客文章”
